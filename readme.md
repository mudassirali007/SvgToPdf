# Installation
```bash
1. git clone https://github.com/theharoonali/SvgToPdf.git
```
2. goto Project Folder. By default folder name is SvgToPdf

```bash
3. npm install
```
```bash
4. node app.js
```
5. Goto http://localhost:3000/ 
6. Paste SVG's
